Demostración de código JavaScript.

Programa que resuelve serie de Fibonacci de un número N de forma recursiva: 
function calcularFibonacci(n) {
    if (n < 2) {
        return n;
    } else {
        return calcularFibonacci(n - 1) + calcularFibonacci(n - 2);
    }
}
